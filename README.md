# TOEIC単語帳 (toeic-tango)

TOEIC学習用のかんたん単語帳アプリ。フラッシュカードで「覚えた / 苦手」を自動で振り分け、あとから一覧で見返せる。HTML/CSS/JS の静的アプリで GitHub Pages に公開。

## 機能
- 📇 **フラッシュカード**: 英単語 → タップで意味表示 → 「覚えた / まだ」で振り分け
- 🗂 **自動振り分け**: 「覚えた / 苦手 / 未学習」に自動分類し、一覧タブで見返せる
- 🎯 **学習デッキ**: おすすめ（苦手＋未学習優先）/ 苦手 / 未学習 / 全部、シャッフル出題
- ➕ 単語の追加・編集・削除
- 📥 **まとめて取り込み**: `英単語[タブ/カンマ]意味` を貼り付けて一括登録
- 📊 進捗表示（覚えた数 / 総数）
- 📱 PWA対応（ホーム画面に追加・オフライン起動可）。データは端末内 localStorage に保存

## 同梱単語について
同梱のスターター単語（約120語）は、TOEIC頻出の一般的なビジネス英単語を**独自にまとめたもの**で、特定の市販単語集からの転載ではありません。自分の単語は追加・取り込みで自由に増やせます。

## 開発プレビュー

node server.js   # http://localhost:5190
